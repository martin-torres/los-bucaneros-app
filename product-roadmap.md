# Los Bucaneros — Product Roadmap

---

## Tier 1: Core (DONE — live now)

| Feature | What It Does |
|---|---|
| QR code menu + PWA | Customer scans, browses, orders |
| Cart + checkout | Item selection, quantities, modifiers |
| Structured address | Street, colonia, details |
| Telegram notification | Full order with Maps link + cambio |
| Kitchen view | Real-time status board |
| Customer tracking | Status pipeline (Recibido → Entregado) |
| Cash + cambio calc | Pay amount + change display |
| Admin module | Menu edits, settings, promos |
| Data view | Order history, basic analytics |

**Tier 1 is the freebie.** Every restaurant gets this. The 5% covers hosting + support.

---

## Tier 2: Essential Growth ($3k–$5k dev — BUILD NEXT)

| Feature | Dev Cost | Value to Owner |
|---|---|---|
| **WhatsApp Business API integration** — orders from WhatsApp auto-enter the system without a human typist | $1,500–$2,500 | Eliminates #1 pain point (WhatsApp time sink) |
| **Customer database with export** — names, phones, addresses, order history, CSV export | $500–$1,000 | Direct marketing channel, $5k–$10k/mo value |
| **One-tap reorder** — regulars repeat last order in 30 seconds | $800–$1,200 | Captures 10–20% more orders from existing customers |

**Why build this first:** WhatsApp integration kills the #1 ranked pain point (scored 24/30). It makes the 5% feel like a robbery in your favor.

---

## Tier 3: Premium ($8k–$15k — months 3–6)

| Feature | Dev Cost | Revenue Model |
|---|---|---|
| **Web Push notifications** — browser push when order status changes | $1k–$1.5k | Included |
| **Delivery time ETA** — estimated times from cook + delivery history | $1.5k–$2.5k | Included |
| **Online payment gateway** (Stripe, Conekta, Clip) — collect at order time, not delivery | $5k–$8k | **Take 1–2% on top** |
| **Multi-location dashboard** — all locations on one screen | $2k–$3k | Unlocks Premium tier pricing |

---

## Tier 4: Ecosystem ($15k–$30k — months 6–12)

| Feature | Dev Cost | Revenue Model |
|---|---|---|
| **Inventory tracking** — what sold, what's left, auto-reorder | $3k–$5k | Premium tier only |
| **Automated loyalty program** — points, birthday offers, visit rewards | $2k–$4k | Premium tier only |
| **SMS/WhatsApp marketing campaigns** — blast offers to database | $2k–$3k | Per-campaign fee or monthly sub |
| **Staff scheduling add-on** — shift management, no-show tracking | $5k–$8k | Low priority |

## Adjacent Revenue Streams

| Stream | Revenue Per Client | Dev Cost | Priority |
|---|---|---|---|
| **Printed QR table tents** — design + print + deliver | $200–$500 one-time | Outsource | 🥇 Quick win |
| **Menu photography** — professional food photos | $1k–$3k per shoot | Outsource | 🥇 Quick win |
| **Payment gateway cut** — 1–2% on top of transaction | $2.5k–$5k/mo at $250k volume | $5k–$8k | 🥈 Months 3+ |
| **Marketing campaigns service** — run their campaigns for them | $2k–$5k/mo | $2k–$3k | 🥉 Months 6+ |
| **Multi-location premium** — flat fee per location | $3k–$5k/mo per chain | $2k–$3k | 🥉 Months 6+ |

---

## What to Build Next (Highest ROI)

**1. WhatsApp API integration.** Kills the #1 pain point. Makes the 5% transaction fee feel cheap. Estimated dev: $1.5k–$2.5k.

**2. Payment gateway.** 1–2% on top of every transaction creates a second revenue layer without changing your base price. $250k/mo volume → $2.5k–$5k/mo extra to you.

**3. Customer database + one-tap reorder.** Low dev cost ($1.5k–$2.2k combined), delivers the database value prop you're selling.
